# String Puppetry Master Registry

This registry documents the regional schools, lineages, and artistic techniques of traditional Chinese string puppetry (提线木偶 / 木偶戏).  
String puppetry is a single master tradition with deep regional diversity: carving styles, string systems, articulation complexity, musical accompaniment, dialects, and narrative forms vary dramatically across China.

## Purpose

The goal of this contract is to preserve authentic string-puppet knowledge in a permanent, verifiable, and censorship‑resistant format.  
Each entry records a real workshop, lineage, or regional school, including:

- region  
- lineage or workshop  
- style name  
- materials  
- manipulation technique  
- performance traits  
- cultural uniqueness  

## Why this registry matters

Chinese string puppetry is one of the most technically sophisticated puppetry traditions in the world.  
Its regional schools differ as much as celadon kilns, embroidery, or shadow puppetry:

- Fujian: highly articulated puppets with 16–32 strings  
- Zhejiang: elegant, slender puppets inspired by Yue opera  
- Guangdong: large Cantonese puppets with opera staging  
- Sichuan: expressive faces with movable eyes and mouth  
- Jiangsu: refined carving and classical narratives  
- Hunan: dramatic, martial, powerful movement  

By recording these variations on‑chain, the registry protects cultural memory and supports future research, education, and artistic revival.
